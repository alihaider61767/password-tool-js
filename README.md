# 🔐 Password Tool (JavaScript)

This project is a simple web-based password tool built using Vanilla JavaScript.  
It allows users to generate secure passwords and check the strength of any password.

The goal of this project is to practice JavaScript logic, random generation, and validation techniques.

## 🚀 Features

- Generate random passwords (8–20 characters)
- Includes uppercase, lowercase, numbers, and special characters
- Check password strength instantly
- Clean and responsive interface

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript (Vanilla JS)

## ▶️ How to Run

1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in your browser

No additional installation is required.
